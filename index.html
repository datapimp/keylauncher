KeyLauncher.js

Keylauncher is a small javascript library that allows you to trigger the execution of a block of javascript code along with the optional loading of external javascript and css resources, in response to certain key sequences or keystrokes.

It is perfect for embedding development / support / diagnostic tools in websites, but only loading the dependencies required to run them when you want, and not making your users load them.

See it in action:

Type 'sup' 

KeyLauncher.onSequence "sup", ()->
  alert('sup')

expensiveFunction = ()->
  # do some stuff which requires on expensive libraries
  # like jquery, underscore, backbone, etc

KeyLauncher.on "ctrl+j", expensiveFunction, requires:["//my-cdn/dependencies.js"]
